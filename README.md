https://rbuetzer.github.io/kinder/

## Development

Install the dependencies

```
yarn install
```

Run the project

```
yarn run dev
```

The app is now served at http://localhost:1234

Run the tests

```
yarn run test
```

Deploy to Github Pages

```
yarn run deploy
```
